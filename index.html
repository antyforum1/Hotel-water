02

Открытие разговора с ЛПР

Первые 30 секунд — говорим коротко и уверенно

📞 ПРИВЕТСТВИЕ (ВЫУЧИТЬ НАИЗУСТЬ)

— Добрый день, [Имя]! Меня зовут [Ваше имя], компания [Название], мы поставляем питьевую воду для отелей и ресторанов Краснодарского края.


Звоню не с предложением — мы сейчас изучаем, как устроены закупки воды в отелях вашего формата, чтобы сделать продукт под реальные потребности рынка. Вы не против ответить на 3–4 коротких вопроса? Это займёт буквально 5 минут.

Пауза. Ждём «да». Не продолжаем, пока не получим разрешение говорить.

🧠

Почему «изучаем рынок»: снижает защитную реакцию. Люди охотно делятся информацией, когда не чувствуют давления продажи.

03

Вопросы-разведка

Задаём по одному, слушаем и записываем

📋

Порядок важен: сначала нейтральные вопросы (что покупают → объём), потом цена. Цену спрашиваем только после контакта — не в лоб.

ВОПРОС 1 — ЧТО ЗАКУПАЮТ

📞 ВЫ:

— Вы закупаете воду в бутылках для гостей — в номера, или только для ресторана / кухни? Или и то, и другое?

Уточните: «А кулеры используете? 19-литровые бутыли?»

💡 СЛУШАЕМ

Форматы: 0.33 / 0.5 / 1.0 / 1.5 л · 19 л кулерные · Брендированные с логотипом · Наличие ресторана/баров

ВОПРОС 2 — ОБЪЁМ

📞 ВЫ:

— Примерно сколько бутылок берёте в месяц — хотя бы порядок цифры? Сотни, тысячи?

Можно добавить: «Просто понять масштаб — нам важно для логистики»

💡 ОРИЕНТИРЫ

до 50 номеров → ~300–800 шт/мес · 50–150 → ~800–3 000 · 150–300 → ~3 000–10 000 · 300+ → от 10 000

ВОПРОС 3 — ПОСТАВЩИК

📞 ВЫ:

— С каким поставщиком работаете сейчас — крупная федеральная марка или местный производитель?

⚡ ВОПРОС 4 — ЦЕНА (КЛЮЧЕВОЙ)

⚡ ВАРИАНТ А — МЯГКИЙ:

— Примерно в каком диапазоне вы сейчас закупаете — до 15 рублей за бутылку, или дороже?

Называем ориентир — метод альтернативы, помогает ответить не раскрывая «секрет»

💬 ВАРИАНТ Б — ПРЯМОЙ:

— Понимаю, что закрытая информация, но хотя бы порядок — вы платите за 0.5л условно 10 рублей или 25 рублей? Нам важно понять, с каким ценовым сегментом работаете.

ВОПРОС 5 — БОЛИ

📞 ВЫ:

— Есть ли что-то, что не устраивает в текущей схеме? Может, доставка с задержками, или минимальная партия слишком большая?

📋 ПОШАГОВАЯ ИНСТРУКЦИЯ

1

Откройте sheets.google.com и создайте новую таблицу. Назовите её «Звонки — Поставка воды». В первой строке заголовки создадутся автоматически.

2

В таблице нажмите Расширения → Apps Script. Откроется редактор кода. Удалите всё что там есть.

3

Вставьте код ниже (нажмите кнопку «Скопировать код») и нажмите Сохранить проект (Ctrl+S).

const SHEET_NAME = "Звонки";


function doPost(e) {

  try {

    const data = JSON.parse(e.postData.contents);

    const ss = SpreadsheetApp.getActiveSpreadsheet();

    let sheet = ss.getSheetByName(SHEET_NAME);

    

    if (!sheet) {

      sheet = ss.insertSheet(SHEET_NAME);

      const headers = [

        "Дата","Время","Менеджер","Отель","Контакт",

        "Результат","Объём (шт/мес)","Цена (руб/шт)",

        "Поставщик","Email","Следующий шаг",

        "Длительность","Боли / Примечания"

      ];

      sheet.getRange(1, 1, 1, headers.length).setValues([headers]);

      sheet.getRange(1, 1, 1, headers.length)

        .setBackground("#1a3a5c").setFontColor("#ffffff")

        .setFontWeight("bold");

      sheet.setFrozenRows(1);

    }

    

    const row = [

      data.date, data.time, data.manager,

      data.hotel, data.contact, data.result,

      data.volume, data.price, data.supplier,

      data.email, data.next, data.duration,

      data.notes

    ];

    sheet.appendRow(row);

    

    const lastRow = sheet.getLastRow();

    if (lastRow % 2 === 0) {

      sheet.getRange(lastRow, 1, 1, row.length)

        .setBackground("#f0f4f8");

    }

    

    return ContentService

      .createTextOutput(JSON.stringify({status:"ok", row: lastRow}))

      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {

    return ContentService

      .createTextOutput(JSON.stringify({status:"error", message: err.toString()}))

      .setMimeType(ContentService.MimeType.JSON);

  }

}


function doGet(e) {

  return ContentService

    .createTextOutput(JSON.stringify({status:"ok", message:"Webhook активен"}))

    .setMimeType(ContentService.MimeType.JSON);

}

Вставьте скопированный URL в поле выше и нажмите Проверить. Если всё ок — нажмите Сохранить. Готово! 🎉
